1 Load in time domain

MATLAB usually stores time signals in a matrix variable where columns are signals and rows are their values over time.
For the purposes of the example, three uncorrelated narrowband signals with a length of 200,000 samples and a sampling frequency of 500 Hz were prepared and stored in example_data.mat file.
e_data = load('example_data.mat');
disp(e_data)
s1: [200000×3 double] s2: [200000×3 double] s3: [200000×3 double] s_sampling_freq: 500
Take the 3 columns signal s1, s2 or s3 as the three independent loads varying with time.
Let's see what these signals look like by plotting them out.
L = [100 100 100].*e_data.s3; % load vector L(t)
L_sampling_freq = e_data.s_sampling_freq;
 
plot(1/L_sampling_freq*(1:size(L,1)), L)
xlabel('time, s'), ylabel('load')
legend('L_{1}(t)','L_{2}(t)','L_{3}(t)')

2 Load in frequency domain

Since we have define the load as time signals, the Welch method is used to obtain the PSD matrix of load vector . For this purpose, we use the cpsd(...) function, which allows us to estimate the power spectral density matrix using the Welch method.
[GL, f] = cpsd(L, L, hanning(2^10), 2^9, [], L_sampling_freq, "mimo");
 
% show the PSD matrix
figure

end
end

5 Indicators and measures for multiaxiality

At this point it is worth mentioning the parameters describing the stress state in the sense of multiaxiality.

5.1 Stress Triaxiality

Stress triaxiality T is defined as the ratio of the mean (hydrostatic) stress to the von Mises equivalent stress
where:
  • or is the mean stress (hydrostatic stress),
  • is the von Mises equivalent stress.
Significance:
  • : Pure shear stress (no hydrostatic component).
  • : Tensile-dominated stress state.
  • : Compressive-dominated stress state.
  • : Hydrostatic stress state (no deviatoric components).
Stress triaxiality is crucial for understanding ductile fracture, as it influences void nucleation and growth in materials.
% container for the indicators
multIndicators=[];
 
Sm = (S(:,1) + S(:,2) + S(:,3))./3;
Seq = sqrt(S(:,1).^2 + S(:,2).^2 + S(:,3).^2 - S(:,1).*S(:,2) - S(:,2).*S(:,3) - S(:,3).*S(:,1) +...
3*S(:,4).^2 + 3*S(:,5).^2 + 3*S(:,6).^2);
multIndicators.T = Sm./Seq;

5.2 Stress Biaxiality Indicatior

The BI is calculated as the smaller principal stress divided by the larger principal stress, excluding the principal stress nearest zero.
where σ₁ = major principal stress, σ₂ = minor principal stress (σ₃ is not used in the classical surface-fatigue biaxiality indicator).
Significance:
  • : Represents pure shear, where the stresses are acting tangentially and equally in two perpendicular directions.
  • : Indicates a uniaxial stress state, where the stress is acting in only one direction.
  • : Indicates a pure biaxial stress state, where two non-zero principal stresses are present in a plane.
% Find the nearest-to-zero value from each row
[~, indToRemove] = min(abs(S123'),[],1);
 
% Remove the nearest-to-zero value from each row
S12 = zeros(size(S123,1), 2);
for i = 1:size(S12,1)
row = S123(i,:);
row(indToRemove(i)) = []; % remove the selected element
if abs(row(1)) > abs(row(2))
S12(i,:) = row;
else
S12(i,:) = row([2,1]);
end
end
% smaller principal stress divided by the larger principal stress
multIndicators.BI = S12(:,2)./S12(:,1);
Now we can plot the parameters and computing the mean of them.
table(mean(multIndicators.T), mean(multIndicators.BI), 'VariableNames', {'Stress Triaxiality','Biaxiality Indicatior'})
