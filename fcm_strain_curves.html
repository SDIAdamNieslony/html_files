1 Introduction

In fatigue analysis, two groups of strain curve types are commonly used to describe the behavior of a material under cyclic loading. The first group is concerned with the initiation of fatigue cracks, the so-called strain-life curves. These curves link the strain amplitude with the number of cycles to crack initiation. The second group is devoted to the description of the relationship between stress and strain under cyclic loading. Both groups contain several mathematical models for modeling curves and methods for their determination. In the strain-life group, the Manson-Coffin-Basquin (MCB) model is common and will be presented, while in the stress-strain group, the Ramberg-Osgood (RO) model will be presented. Details can be read among others in [1].

2 Strain-life curve according Manson-Coffin-Basquin model (MCB)

Manson and Coffin related the number of cycles to crack initiation to the amplitude of plastic strain. Together with the modified form of the Basquin equation, this work led to the known equation of the strain-life curve
Eq. (1)
where the total strain amplitude is divided into elastic and plastic components. It is easy to see that in the MCB equation there are four fatigue constants:
- fatigue ductility coefficient,
- fatigue strength coefficient,
b - fatigue strength exponent,
c - fatigue ductility exponent,
and one static constant, i.e. Young's modulus E. If these constants are known, a graph with strain curves can be drawn, which is most often presented in a double-logarithmic system.
E = 210000; % Young modulus, MPa
sigpf = 800; % fatigue strength coefficient, MPa
epspf = 0.8; % fatigue ductility coefficient, mm/mm
b = -0.2; % fatigue strength exponent, no units
c = -0.9; % fatigue ductility exponent, no units
Nf = logspace(1,6,100);
eae = sigpf/E*(2*Nf).^b;
eap = epspf*(2*Nf).^c;
 
figure
loglog(Nf, eap, 'r', Nf, eae, 'b', Nf, eap+eae, 'k')
xlabel('N_{f}, cycle'), ylabel('\epsilon_{a,p}, \epsilon_{a,e}, \epsilon_{a,t}')
legend('\epsilon_{a,p}','\epsilon_{a,e}','\epsilon_{a,t}')

np = 0.1; % cyclic strain hardening exponent
Kp = 900; % cyclic strength coefficient
Sa = linspace(0,600);
eae = Sa./E;
eap = (Sa./Kp).^(1/np);
 
figure
plot(eap, Sa, 'r', eae, Sa, 'b', eap+eae, Sa, 'k')

4 Reading values from strain curves

Determining the total strain amplitude is simple using both the MCB and RO formulas. It is determined by substituting the number of cycles for MCB and the stress amplitude for RO, respectively. However, we encounter a problem if we want to determine the inverse function. Both the MCB formula and the RO formula consist of two terms and it is difficult to determine analytically the inverse functions, i.e. for MCB a function that determines the number of cycles for a given total strain amplitude
Eq. (3)
and for RO a function that determines the stress amplitude for a given total strain amplitude
Eq. (4)
In both cases we can use two methods of determining the value:
1) numerically,
2) by interpolation.

4.1 Numerically

The numerical method consists in using the function fzero to find the number of cycles for a given total strain amplitude . The problem here is to define the function that is the input to the optimization procedure. Below is an example of an MCB curve using this function, along with comments.
% defining inline function based on MCB and strain amplitude, the y
% function is constructed in such a way that it returns a value of 0 for
% the number of cycles occurring at a given total strain amplitude
y1 = @(Nf, eat, B, b, C, c) B.*(2*Nf).^b + C.*(2*Nf).^c - eat;
opt = optimset('fzero');
 
eatGiven = 0.01;
NfFound = fzero(y1, [1 1e100], opt, eatGiven, sigpf/E, b, epspf, c);
disp(table(eatGiven, NfFound))
eatGiven NfFound ________ _______ 0.01 76.881
Similarly we can determine the stress amplitude from the RO function.
% defining inline function based on RO and strain amplitude, the y
% function is constructed in such a way that it returns a value of 0 for
% the stress amplitude occurring at a given total strain amplitude
y2 = @(Sa, eat, E, Kp, np) Sa./E + (Sa./Kp).^(1/np) - eat;
opt = optimset('fzero');
 
eatGiven = 0.01;
saFound = fzero(y2, [0 1e3], opt, eatGiven, E, Kp, np);
disp(table(eatGiven, saFound))
eatGiven saFound ________ _______ 0.01 550.85

4.2 Using Interpolation

When determining fatigue life, it is often necessary to determine the number of cycles from the MCB characteristic or the stress amplitude from the RO curve multiple times. This occurs, for example, in the fatigue damage summation procedure when analyzing time history with variable amplitude, where fatigue damage summation is used. In this case, the numerical method, which is very accurate, becomes slow and the interpolation method can be used. This method is less accurate than the numerical one, but very fast. For the interpolation method to be effective, it is best to logarithm the values and interpolate onto the variables linearized in this way. Examples for several values are provided below.
% some total strain aplitudes
eatGiven = [0.1 0.09 0.07 0.05 0.03 0.001 0.0005 0.0002 0.0001]';
% function for MCB curve
eN = @(Nf) sigpf/E*(2*Nf).^b + epspf*(2*Nf).^c;
% limits on the number of cycles for the search area
NfLimits = [0.5 10^10];
NfInterp = logspace(log10(NfLimits(1)), log10(NfLimits(2)), 1000);
aInterp = eN(NfInterp);
NfFoundLog = interp1(log(aInterp), log(NfInterp), log(eatGiven), 'linear','extrap');
 
% plot found points
figure
plot(log(NfInterp), log(aInterp), 'b', NfFoundLog, log(eatGiven), 'ro')
xlabel('log(N_f), cycle')
ylabel('log(\epsilon_{at}), mm/mm')
% print the amplitudes and corresponding cycles
NfFound = exp(NfFoundLog);
disp(table(eatGiven, NfFound))
eatGiven NfFound ________ __________ 0.1 5.1768 0.09 5.8332 0.07 7.7613 0.05 11.402 0.03 20.579 0.001 2909.2 0.0005 22307 0.0002 1.2968e+06 0.0001 4.024e+07

5 Strain curves from experimental data

5.1 Conventional method

Fatigue tests of laboratory specimens under controlled strain amplitude with zero mean stress allow obtaining three values: stress amplitude, strain amplitude and number of cycles to fatigue failure. These values are used to determine the strain fatigue curve and the cyclic stress-strain curve. Below I will show how to obtain these constants using the conventional method, which consists in approximating suitably prepared data with three straight lines. First, however, we will define the points based on data for StE 690 low-alloy steel [2].
% Experimental data for StE 690 Low-Alloy Steel, in columns:
% stress amplitude MPa; strain amplitude %; number of cycles;
expData = [742 3.200 95
730 3.200 100
675 1.600 280
675 1.600 300
594 0.800 1090
595 0.800 1170
576 0.600 2980
578 0.600 3230
554 0.500 4284
534 0.350 13900
486 0.300 20120
529 0.300 33000
492 0.280 44000
530 0.255 82600
482 0.225 1150000
475 0.224 1500000];
 
% Young's modulus for elastic strain
expYoungModulus = 214000; % MPa
 
% plot raw data
figure
loglog(expData(:,3), expData(:,2),'o')
axis([1e1 1e7 0.1 10])
xlabel('number of cycle, N_f')
ylabel('strain amplitude, ea, mm/mm')
With this data set including stress amplitude, the Young’s modulus and the total strain amplitude the plastic part of the strain amplitude can be calculated
Eq. (5)
Then all possible variables will be defined and additionally the plastic strain amplitude level below which the experimental points will not be analyzed for the plastic strain curve.
% Young’s modulus MPa
E = expYoungModulus;
% stress amplitudea MPa
sa = expData(:, 1);
% strain amplitude mm/mm
ea = 0.01*expData(:, 2);
% number of cycles
Nf = expData(:, 3);
% elastic strain amplitude mm/mm
eae = sa/E;
% plastic strain amplitude mm/mm
eap = ea - eae;
Caution is needed here! We are working with experimental data, and it may happen that after subtracting the elastic part from the total strain amplitude, Eq. (5), we obtain a very small value or even a negative one. This often occurs at a high number of cycles, where plastic deformation no longer plays a significant role. Therefore, for the purpose of determining material constants, a threshold is established for the relevance of the plastic strain contribution in the approximation.
% plastic strain limit value mm/mm
e0 = 0.0001;
% indexes to data whose eap value > e0
ind = eap>e0;
To determine the six fatigue constants, three independent linear regressions must be performed. In this case we will use the polyfit function, but first we need to linearize the appropriate functions according to the table below.
% initiation of variable 'w' for all results
w = [];
 
% calculating Epf, c
x = log(2*Nf(ind));
y = log(eap(ind));
p = polyfit(x, y, 1);
w.c = p(1);
w.epspf = exp(p(2));
 
% calculating Spf, b
x = log(2*Nf);
y = log(eae);
p = polyfit(x, y, 1);
w.b = p(1);
w.sigpf = exp(p(2))*E;
 
% calculating Kp, np
x = log(eap(ind));
y = log(sa(ind));
p = polyfit(x, y, 1);
w.np = p(1);
w.Kp = exp(p(2));
 
figure
tiledlayout(2,1)
 
nexttile
loglog(Nf, ea, 'ok', Nf(ind), eap(ind), 'sr', Nf, eae, 'xb')
Nf_wyk = logspace(1,8,100);
eas_wyk = w.sigpf/E*(2*Nf_wyk).^w.b;
eap_wyk = w.epspf*(2*Nf_wyk).^w.c;
hold on
loglog(Nf_wyk, eas_wyk+eap_wyk,'k',Nf_wyk, eap_wyk,'r',Nf_wyk, eas_wyk,'b')
hold off
xlabel('number of cycle, N_{f}')
ylabel('strain amplitude, \epsilon_{a}, \epsilon_{ap}, \epsilon_{ae}')
legend('experiment \epsilon_{a}', 'experiment \epsilon_{ap}', 'experiment \epsilon_{ae}',...
'function \epsilon_{a}','function \epsilon_{ap}','function \epsilon_{ae}')
 
nexttile
saPlot = linspace(0, 800, 100);
eaPlot = saPlot./E + (saPlot./w.Kp).^(1/w.np);
plot(ea(ind),sa(ind),'ok', eaPlot, saPlot, 'k')
xlabel('strain amplitude \epsilon_{a}, mm/mm')
ylabel('stress amplitude \sigma_{a}, MPa')
legend('experiment','function')
% displaying Young's modulus and 6 fatigue constants
disp(table(E,w.sigpf,w.epspf,w.b,w.c,w.Kp,w.np,'VariableNames',["Young Mod.","Sig_pf","Eps_pf","b","c","Kp","np"]))

5.2 K' and n' from compatibility equations

There are so-called compatibility equations that relate the coefficients and exponents that appear in the Ramberg–Osgood and Manson–Coffin–Basquin formulas. This allows one to determine the coefficient and exponent on the stress–strain curve directly from the strain–life curve. Note! These coefficients may not be equal to those determined based on a approximation. More on this can be found in [1].

Eq. (7)
% definition of n' and K' according Eqs. (6) and (7)
w.npComp = w.b/w.c;
w.KpComp = w.sigpf/(w.epspf^(w.b/w.c));
