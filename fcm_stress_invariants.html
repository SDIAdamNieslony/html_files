Introduction

Stress invariants are scalar quantities derived from the stress tensor that remain unchanged under coordinate transformations. They provide a basis for evaluating the stress state irrespective of the coordinate system which is useful for fatigue analysis under complex multiaxial loading. Stress invariants are basis for various fatigue criteria and yield criteria, such as von Mises, Tresca, Dang Van and Crossland. If the stress tensor is defined as:

1 Calculation of Stress Invariants

For a given stress tensor it is easy to calculate the stress state invariants according to the formulas given earlier.
% Define the stress tensor components (units: MPa, for example)
sigma_xx = 120;
sigma_yy = 80;
sigma_zz = 60;
tau_xy = 15;
tau_yz = 10;
tau_zx = 5;
 
% Stress Tensor
sigma = [sigma_xx, tau_xy, tau_zx;
tau_xy, sigma_yy, tau_yz;
tau_zx, tau_yz, sigma_zz];
 
% First Invariant (Mean Stress)
I1 = trace(sigma);
 
% Deviatoric Stress Tensor
sigma_mean = (1/3) * I1 * eye(3);
s = sigma - sigma_mean;
 
% Second Invariant
I2 = 0.5 * (I1^2 - trace(sigma * sigma));
 
% Third Invariant (Determinant)
I3 = det(sigma);
 
disp(table(I1,I2,I3))
I1 I2 I3 ___ _____ _______ 260 21250 5.5e+05
Of course, definitions using stress state components also give correct results. Using the second invariant of the stress tensor the von Mises stress can be computed
 
% von Mises Equivalent Stress
sigma_von_mises = sqrt(3/2 * sum(sum(s.^2)));
