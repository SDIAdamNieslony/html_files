Introduction

In multiaxial fatigue, multiaxial fatigue failure criteria are used to determine equivalent stresses or strains in terms of fatigue. These criteria are essential for the use of fatigue curves, which are prepared for uniaxial loading conditions. In statics the von Mises criterion is very popular and is also often used in fatigue. 1 In-phase constant amplitude loading

The constant-amplitude in-phase load is the simplest case of multiaxial loading. In this case, the equivalent stress according to the von Mises hypothesis can be determined using only the amplitudes at which the maximum values of the time histories converge.
For clarity, the example will be analyzed in a plane stress state, i.e. only the three components of the tensor, will be considered in the analysis.
% defining the amplitudes
Sxx = 100;
Syy = 30;
Txy = 50;
% time vector
t = linspace(0, 10, 1000)';
% stress time history
S1 = [Sxx*sin(t) Syy*sin(t) Txy*sin(t)];
% Eq. von Mises stress according Eq.(2)
sm1 = sqrt(S1(:,1).^2 - S1(:,1).*S1(:,2) + 3*S1(:,3).^2);
 
figure
h = plot(t, [S1 sm1 max(sm1)*ones(length(t),1)]);
h(5).LineStyle = '--';
xlabel('time, s')
ylabel('stress, MPa')
legend('Sxx','Syy','Txy','von Mises','max(von Mises)')

2 Out-off-phase constant amplitude loading

The von Mises criterion for a stress state of constant amplitude with a phase shift is also often used. In this case, von Mises stress history is searched for to find the maximum stress. If necessary, we additionally have information on the multiaxiality of the stress based on appropriately defined parameters.
% add phase shifts to the amplitudes defined earlier
SxxPhi = 0;
SyyPhi = pi/2;
TxyPhi = pi/3;
% stresses
S2 = [Sxx*sin(t + SxxPhi) Syy*sin(t + SyyPhi) Txy*sin(t + TxyPhi)];
% Eq. von Mises stress according Eq.(2)
sm2 = sqrt(S2(:,1).^2 - S2(:,1).*S2(:,2) + 3*S2(:,3).^2);
 
figure
h = plot(t, [S2 sm2 max(sm2)*ones(length(t),1)]);
h(5).LineStyle = '--';
xlabel('time, s')
ylabel('stress, MPa')
legend('Sxx','Syy','Txy','von Mises','max(von Mises)')
It is easy to see that in both the in-phase and out-of-phase time histories the von Mises stress shows a doubled frequency. In fatigue analysis this fact is not taken into account (doubling the number of loading cycles), and the maximum value of the equivalent von Mises stress is used, treating this value as the equivalent amplitude with the loading frequency of the stress components.
Interestingly, for the phase shift load case, the calculations can be performed on complex numbers. Bonte et al. [2] took advantage of this possibility and took into account the phase shift with respect to each component of the stress state. To see how to do this, see the code below and read the provided literature.
% von Mises stress according Bonte et al. [2]
% "mean stress" of the von Mises time history
m23_l = 0.5*(Sxx^2 + Syy^2 + 3*Txy^2 - Sxx*Syy*cos(SxxPhi-SyyPhi));
% "fluctuating part" of von Mises stress
m23_r = 0.5*(Sxx^2*exp(2*1i*SxxPhi) + Syy^2*exp(2*1i*SyyPhi) + 3*Txy^2*exp(2*1i*TxyPhi) + ...
-Sxx*Syy*exp(1*1i*(SxxPhi + SyyPhi)));
 
% phase shift of the von Mises stress complex number
BontePhi = 0.5*angle(m23_r);
% and von Mises stress ampitude
BonteAmp = sqrt(m23_l + abs(m23_r));
 
% display obtained values:
disp(table(sqrt(m23_l),BonteAmp,BontePhi*180/pi,'VariableNames',["von Mises mean","von Mises ampl.","von Mises phase"]))
von Mises mean von Mises ampl. von Mises phase ______________ _______________ _______________ 95.917 111.33 16.578
% plot the results
figure
h = plot(t, [S2 sm2 sqrt(m23_l)*ones(length(t),1) BonteAmp*sin(t + BontePhi)]);
h(5).LineStyle = "--";
xlabel('time, s')
ylabel('stress, MPa')
legend('Sxx','Syy','Txy','von Mises','von Mises mean','von Mises (Bonte)')

3 Random loading

3.1 Signed von Mises stress

The main problem in using the von Mises stress for fatigue calculations is that it always takes positive values. Therefore, there have been proposals to add a sign to the calculated values. In a signed von Mises stress, a positive or negative sign is associated with the von Mises stress to reflect the dominant nature of the stress state, e.g. tension or compression. There are various ways to define this sign, but the most common methods include:
  • Using the maximum principal stress: the sign of the von Mises stress is determined by the sign of the maximum principal stress.
  • From hydrostatic stress: the sign can also be determined using the hydrostatic stress stress tensor.
Let us first see how this works for the phase-shifted stresses analyzed earlier.
% we need principal stresses from S2
% average stress
sigma_avg = (S2(:,1) + S2(:,2))./2;
% radius of Mohr's circle
R = sqrt(((S2(:,1) - S2(:,2))./2).^2 + S2(:,3).^2);
pStresses = [sigma_avg+R sigma_avg-R];
 
% we are looking for signs
vmSign = zeros(length(sm2), 1);
for i=1:length(vmSign)
if abs(pStresses(i,1)) > abs(pStresses(i,2))
vmSign(i) = sign(pStresses(i,1));
else
vmSign(i) = sign(pStresses(i,2));
end
end
 
figure
plot([S2 sm2 sm2.*vmSign])
xlabel('time, s')
ylabel('stress, MPa')
legend('Sxx','Syy','Txy','von Mises','von Mises signed')
As can be seen, the time course of the von Mises stress shows discontinuities at the point of sign change, this is even more visible in random stress.

3.2 Standard deviation of von Mises stress
