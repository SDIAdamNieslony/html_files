1 Introduction

The SWT criterion uses the strain amplitude and the maximum stress perpendicular to the critical plane when defining the fatigue parameter. Since there is a product of these quantities, it is considered as an energy criterion. The SWT criterion is usually applied to brittle or hard materials or situations where tensile cracks dominate and the mean stress plays a significant role. The SWT damage parameter is given by:

Initially, the SWT criterion was a uniaxial model proposed for load conditions where tension dominated. Its implementation on the critical plane allows it to deal much more effectively with real multiaxial fatigue situations and also works correctly for geometrically complex structures. 3 Multiaxial Loading with Mean Stress and Critical Plane Concept

3.1 Constant Amplitude Loading

To apply the Smith-Watson-Topper criterion in a multiaxial stress condition, the concept of a critical plane is used. The SWT parameter is evaluated on different material planes, and the critical plane is identified as the one where the maximum fatigue damage occurs. This parameter is calculated in the direction normal to each potential critical plane. Consider the case of a multiaxial load with a nonzero mean stress, where all load components have a constant amplitude and a common frequency.
% Stress amplitudes for [Sxx Syy Szz Txy Tyz Tzx]
Samp = [100 0 0 100 0 0];
% Mean stresses for each component
Smean = [150 0 0 0 0 0];
% Phase shift in degrees
Sphase = [0 0 0 60 0 0];
 
length_time = 4; % Duration in seconds
fs = 1000; % Sampling frequency in Hz
freq = 1; % Frequency of constant amplitude loading (Hz)
 
% Time vector
t = linspace(0, length_time, fs*length_time);
 
% Initialize stress matrix
Sigt = zeros(6, length(t));
 
% Generate stress signals
for i = 1:6
Sigt(i, :) = Smean(i) + Samp(i)*sin(2*pi*freq*t + deg2rad(Sphase(i)));
end
 
% Plot stress components
figure
 
Snames = {'S_{xx}', 'S_{yy}', 'S_{zz}', 'T_{xy}', 'T_{yz}', 'T_{zx}'};
plot(t, Sigt)
xlabel('time, s')
ylabel('stress, MPa')
grid on
legend(Snames)
In the next step, we need to search all possible positions of the critical plane for one that will give the maximum SWT parameter. The simplest way is to generate a mesh on a sphere of radius 1 and use the position as the direction cosines of the unit vectors. Projecting the stresses in this direction will give us the value we are looking for.
% Critical plane possible positions
cpNo = 1000;
cpPoints = generate_spherical_points(cpNo);
 
PSWT = zeros(cpNo, 1);
for i=1:cpNo
Sn = projectionNormalDirection(Sigt, cpPoints(i,:));
% stress max value
Smax = max(Sn);
% stress amplitude
Samp = range(Sn)/2;
% strain amplitudes
epsa = Samp./Ep + (Samp./Kp).^(1/np);
PSWT(i) = Smax*epsa;
end
 
figure
tiledlayout(1,2)
 
nexttile
plot(PSWT)
title('P_{SWT} for each CP position')
 
nexttile
scatter3(cpPoints(:,1),cpPoints(:,2),cpPoints(:,3),8,PSWT,'filled')
title('P_{SWT} as a color on sphere')
grid on, axis equal
Having calculated the values in each direction (opints on the sphere), we select the maximum value and calculate the number of cycles from the fatigue strain curve.
cpPSWT = max(PSWT);
Nf = fzero(swt_fun, [1, 1e20], opt, cpPSWT, Spf, b, epf, c);
disp(table(cpPSWT,Nf))

3.2 Variable Amplitude Loading

Let us consider the general case in which the load components and the stress and strain states can take on any, even random, character. In this case, the biggest challenge is to correctly determine the stress and strain time histories at the material point at which we want to determine the fatigue life. One might say that Neuber's rule should be used as before, but...
The Neuber rule, while primarily developed for constant amplitude loading, can be adapted for variable amplitude loading scenarios, particularly when dealing with elastic-plastic behavior at notch roots. It essentially provides a way to estimate the elastic-plastic stress and strain at the notch tip based on a linear elastic analysis. However, applying it directly to variable amplitude loading requires careful consideration of the load sequence and material response. For example this method does not take into account ratchetting and hardening effects.
The second important task is to determine the maximum stress value and strain amplitude in the direction perpendicular to the critical plane. Since we are dealing with a random load, for this purpose we use a multiaxial version of the rainflow cycle counting. This method allows us to indicate the maximum stress value occurring during the strain cycle period, which was determined based on time histories. After that Miner rule is also used for evaluating damage. This procedure is definitely more complicated than the previous one, in which the load was assumed as of constant-amplitude type. I will be happy to explain this case upon direct inquiry.

Internal Function used in the Live Script

function value = projectionNormalDirection(Sigt, cpPoint)
% Cosines direction for normal vector to the plane
nx = cpPoint(1);
ny = cpPoint(2);
nz = cpPoint(3);
% Evaluatineg the value, clasical projection on vector
value = Sigt(:,1).*nx^2 + Sigt(:,2).*ny^2 + Sigt(:,3).*nz^2 + ...
Sigt(:,4).*2*nx.*ny + Sigt(:,5).*2*ny.*nz + Sigt(:,6).*2*nz.*nx;
end
