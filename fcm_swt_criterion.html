1 Determining the Position of a Plane in 3D Space

The most common and simplest method of determining a plane in three-dimensional space is to use a unit vector normal to that plane. A unit vector n⃗ is defined as:
Eq. (1)
The coordinate of the unit vector is equal to the projection of the vector onto the coordinate axes and the cosines of the angles measured to the corresponding axes. Hence the coordinates of the unit vector are often called direction cosines. This is illustrated in the figure below.
Fig. 1. Directional cosines and coordinates of the normal unit vector indicating the position of the critical plane
In the critical plane fatigue criteria, two values are often defined in the direction normal and tangential to the plane, hence the unit vector lying in the plane and perpendicular to the normal vector must also be known
Eq. (2)
Fig. 2. Normal unit vector and tangential unit vector on the critical plane
Using the normal and tangent vectors in the critical plane, the stresses or strains in these directions are determined using the stress or strain tensors. Let me show this using symbolic operations.
% Symbols declaration:
 
% a) directional cosines for normal and tangent vectors:
% n = [nx ny nz], t = [tx ty tz]
syms nx ny nz tx ty tz real
% b) tensor components
syms Sxx Syy Szz Sxy Syz Szx real
 
% For a selected plane:
% a) vector normal to the plane
n = [nx ny nz];
 
% b) and tangent to the normal vector laying on the plane
t = [tx ty tz];
 
% stress tensor
S = [Sxx Sxy Szx; Sxy Syy Syz; Szx Syz Szz];
disp(S)
Having the versors in the normal and tangential directions on the critical plane we can determine the stresses in these directions using the stress tensor.
% stress in the direction of the n vector (normal to the plane)
Sn = n*S*n';
disp(Sn)
% stress in the direction of the t vector (laying on the plane)
St = n*S*t';
disp(St)
Most multiaxial fatigue criteria using the critical plane concept determine the parameter responsible for fatigue by summing the portion of the normal and shear stresses at a fixed plane. Given a normal and tangent vector in 3D space, one can calculate the stresses in these directions using the stress tensor. The proportion of shear and normal stress can be determined by constant coefficients K and B.
syms K B real
%B=0;
K=0;
% general form of the criterion
crit = B*St + K*Sn;
 
% we extract the coefficients before the tensor components and write them down to a(i)
sig = [Sxx Syy Szz Sxy Syz Szx];
a = cell2sym(cell(6, 1));
for i = 1:6
k = crit;
for j = 1:6
if j ~= i
k = subs(k, sig(j), 0);
else
k = subs(k, sig(j), 1);
end
end
a(i) = simplify(k);
end
disp(a')
Using symbolic calculations, we obtained 6 mathematical expressions in the variable a. For one position of the critical plane, these are constants which, multiplied by the appropriate members of the stress tensor , give the equivalent stress according to the selected criterion.

2 Setting the Critical Plane Position

One of the most important elements when applying multiaxial fatigue criteria based on the concept of a critical plane is determining its location. This location depends on the assumptions of the criterion. Sometimes it is the location that gives the greatest damage value according to the criterion assumptions and sometimes it is the location of the greatest shear or normal stress. Please look into the definition of particular criterion to determine this.
Regardless, the common problem of all criteria is the search for a local maximum. The problem is even greater because there may be many areas with a local maximum or one global maximum extending over a certain area.

Function used in the LiveScript

function value = seq(x, K, B, Sxx, Syy, Szz, Sxy, Syz, Szx)
% Rozpakowanie kosinusów kierunkowych
nx = x(1); ny = x(2); nz = x(3);
tx = x(4); ty = x(5); tz = x(6);
 
% Obliczanie wartości funkcji celu (Seq)
value = Sxx*(K*nx^2 + B*tx*nx) + ...
Syy*(K*ny^2 + B*ty*ny) + ...
Szz*(K*nz^2 + B*tz*nz) + ...
Sxy*(B*(nx*ty + ny*tx) + 2*K*nx*ny) + ...
Szx*(B*(nx*tz + nz*tx) + 2*K*nx*nz) + ...
Syz*(B*(ny*tz + nz*ty) + 2*K*ny*nz);
end
 
function [c, ceq] = constraints(x)
% Rozpakowanie kosinusów kierunkowych
nx = x(1); ny = x(2); nz = x(3);
tx = x(4); ty = x(5); tz = x(6);
 
% Ograniczenia równościowe
c = []; % Brak ograniczeń nierównościowych
ceq = [
nx^2 + ny^2 + nz^2 - 1; % Normalizacja wektora n
tx^2 + ty^2 + tz^2 - 1; % Normalizacja wektora t
nx*tx + ny*ty + nz*tz % Prostopadłość wektorów n i t
];
end
