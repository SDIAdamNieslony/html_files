Critical plane methods

Time domain

All criteria using the concept of critical plane require the determination of this plane in relation to the material. The simplest method is to search the possible positions of the critical plane and determine in which position the fatigue damage is greatest. To do this, we need to generate a set of critical plane positions by specifying the unit vectors to which the plane is perpendicular. To do this, we can use the sphere function where the points on the sphere with radius of 1 are the coordinates of the unit vectors.
[dcx, dcy, dcz] = sphere(50);
mesh(dcx, dcy, dcz)
axis equal
xlabel('X'), ylabel('Y'), zlabel('Z')
Searching for the position of the critical plane by determining the damage in every possible position takes a long time. Therefore, the determination of the critical plane using the maximum variance method is often used. This method consists in determining the maximum value of the variance determined in the same way as defined by this criterion for the location of the critical plane.
% vector of coefficients a for the criterion of the maximum normal stress in the critical plane
a = zeros(length(dcx(:)),6);
a(:,1) = dcx(:).^2;
a(:,2) = dcy(:).^2;
a(:,3) = dcz(:).^2;
a(:,4) = 2*dcx(:).*dcy(:);
a(:,5) = 2*dcy(:).*dcz(:);
a(:,6) = 2*dcz(:).*dcx(:);
 
% covariance matrix from 3D stress vector
covS = cov(S);
disp(covS)
 
% equivalent stress variance for each position of the critical plane
v_eq = zeros(size(a,1),1);
for i = 1:length(v_eq)
v_eq(i) = a(i,:)*covS*a(i,:)';
end
 
% plot the sphere with the color of the variance
surf(dcx,dcy,dcz,reshape(v_eq,size(dcx,1),size(dcx,2)))
axis equal
xlabel('X'), ylabel('Y'), zlabel('Z')
colorbar
We have obtained a sphere with colors representing the variance of the equivalent stress according to the criterion of the maximum normal stress in the critical plane.
In order to determine the fatigue damage according to this criterion, it is necessary to:
  • find the position where the variance reaches the maximum,
  • determine the equivalent stress time history,
  • perform cycle counting using the rainflow method,
  • determine the durability by summing the fatigue damage computed from each stress cycle with Mainer rule.
% find the critical plane position with maximum of variance
v_eq_max_ind = find(v_eq==max(v_eq));
a_fix = a(v_eq_max_ind(1),:);
 
% determine the equivalent stress time history
S_eq = S*a_fix';
plot(S_eq)
% perform the rainflow counting
rainflow(S_eq) % only for plotting the rainflow matrix
rf = rainflow(S_eq); % rf data
c_counts = rf(:,1);
c_ampl = 0.5*rf(:,2);
 
% compute the damage
N = Nf/Saf^(-m)*c_ampl.^(-m);
D = sum(c_counts./N);
T_rf = 1/D*length(S_eq)*s_sampling_time; % time to failure in seconds
 
disp(['Fatigue life: ' num2str(T_rf/(3600*24)) ' days'])

Frequency domain

We will now perform the calculations in the frequency domain. To do this, we first need to determine the stress power spectral density matrix from the stress histories that we read earlier and used for time domain.
% stress PSD matrix
[GS, f] = cpsd(S, S, hanning(2^10), 2^9, [], 1/s_sampling_time, "mimo");
 
% covariance from PSD matrix
covS_f = squeeze(real(trapz(f, GS)));
 
% equivalent stress variance for each position of the critical plane
v_eq_f = zeros(size(a,1),1);
for i = 1:length(v_eq)
v_eq_f(i) = a(i,:)*covS_f*a(i,:)';
end
 
% plot the sphere with the color of the variance
surf(dcx,dcy,dcz,reshape(v_eq_f,size(dcx,1),size(dcx,2)))
axis equal
xlabel('X'), ylabel('Y'), zlabel('Z')
colorbar
% find the critical plane position with maximum of variance
v_eq_f_max_ind = find(v_eq_f==max(v_eq_f));
a_fix_f = a(v_eq_f_max_ind(1), :);
 
% compute the PSD of eqivalent stress
GS_eq = zeros(size(GS,1), 1);
for i=1:length(GS_eq)
GS_eq(i) = a_fix_f*squeeze(real(GS(i,:,:)))*a_fix_f';
end
 
% plot the equvalent PSD
plot(f, GS_eq)
xlabel('frequency, Hz'), ylabel('PSD, MPa^2/Hz')
trapz(f, GS_eq)
psd_mom = zeros(5,1);
for i=1:5 % 5 PSD moments
psd_mom(i) = trapz(f, GS_eq.*f.^(i-1));
end
 
x_pdf_f = linspace(1, sqrt(psd_mom(1))*5, 1000);
y_pdf_f = pdfspectral(psd_mom, x_pdf_f, 'Dirlik');
 
histogram(c_ampl,'Normalization','pdf')
hold on
plot(x_pdf_f, y_pdf_f)
hold off
% expected peaks per unit time
EP = sqrt(psd_mom(5)/psd_mom(3));
 
% compute the damage
N_f = Nf/Saf^(-m)*x_pdf_f.^(-m);
D_f = EP*trapz(x_pdf_f, y_pdf_f./N_f);
 
T_f = 1/D_f; % time to failure in second
 
disp(['Fatigue life (time domain): ' num2str(T_rf/(3600*24)) ' days'])
disp(['Fatigue life (spectral method): ' num2str(T_f/(3600*24)) ' days'])

Eqivalent von Mises stress

The same calculations will be made for the power spectral density determined following the Preumont and Piefort proposition (EMS criterion).
Qmises = [
1 -1/2 -1/2 0 0 0;...
-1/2 1 -1/2 0 0 0;...
-1/2 -1/2 1 0 0 0;...
0 0 0 3 0 0;...
0 0 0 0 3 0;...
0 0 0 0 0 3];
 
% compute the PSD of eqivalent stress
GSMises_eq = zeros(size(GS,1), 1);
for i=1:length(GSMises_eq)
GSMises_eq(i) = trace(Qmises*squeeze(real(GS(i,:,:))));
end
 
psdMises_mom = zeros(5,1);
for i=1:5 % 5 PSD moments
psdMises_mom(i) = trapz(f, GSMises_eq.*f.^(i-1));
end
 
%x_pdf_f = linspace(1, sqrt(psd_mom(1))*5, 1000);
y_pdfMises_f = pdfspectral(psdMises_mom, x_pdf_f, 'Dirlik');
 
histogram(c_ampl,'Normalization','pdf')
hold on
plot(x_pdf_f, y_pdf_f)
plot(x_pdf_f, y_pdfMises_f)
hold off
legend('rainflow','crit. plane','von Mises EMS')
 
% expected peaks per unit time
EPMises = sqrt(psdMises_mom(5)/psdMises_mom(3));
 
% compute the damage
N_f = Nf/Saf^(-m)*x_pdf_f.^(-m);
DMises_f = EPMises*trapz(x_pdf_f, y_pdfMises_f./N_f);
 
TMises_f = 1/DMises_f; % time to failure in second
disp(['Fatigue life (time domain): ' num2str(T_rf/(3600*24)) ' days'])
disp(['Fatigue life (spectral method): ' num2str(T_f/(3600*24)) ' days'])
disp(['Fatigue life (spectral von Mises): ' num2str(TMises_f/(3600*24)) ' days'])
