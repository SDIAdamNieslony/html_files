Introduction

All multiaxial fatigue faliure criteria using the concept of critical plane require the determination of this plane in relation to the material. We also need mathematical tools that will allow us to describe this position in a simple way, without additional numerical operations. The location of the critical plane depends on the type of multiaxial fatigue criterion. However, the method of finding the potential position area is similar for each criterion from this group.
The simplest method involves searching for the potential positions of the critical plane and determining which one corresponds to the maximum fatigue damage, as defined by the selected criterion. To achieve this, we need to generate a set of possible critical plane orientations by specifying the unit vectors perpendicular to the plane. This can be done by using cosine directions, where the points on a unit sphere (radius = 1) represent the coordinates of the unit vectors. Some criteria use a tangent vector lying in the plane, which will also be shown.

Using the normal and tangent vectors in the critical plane, the stresses or strains in these directions are determined using the stress or strain tensors. For practical reasons, a vector representation of the components of the stress or strain tensor is used. Let me show this using symbolic operations.
% Symbols declaration:
 
% directional cosines for normal n and tangent t vectors:
% n = [nx ny nz], t = [tx ty tz]
syms nx ny nz tx ty tz real
 
% tensor components
syms Sxx Syy Szz Sxy Syz Szx real
 
% For a selected plane the vector normal to the plane
n = [nx ny nz];
% and tangent to the normal vector laying on the plane
t = [tx ty tz];
 
% Stress tensor
S = [Sxx Sxy Szx;...
Sxy Syy Syz;...
Szx Syz Szz];
disp(S)
Having the versors in the normal and tangential directions on the critical plane we can determine the stresses in these directions using the stress tensor.
% Stress in the direction of the n vector (normal to the plane)
Sn = n*S*n';
disp(Sn)
% stress in the direction of the t vector (laying on the plane)
St = n*S*t';
disp(St)
Most multiaxial fatigue criteria using the critical plane concept determine the parameter responsible for fatigue by summing the portion of the normal and shear stresses at a fixed plane. Given a normal and tangent vector in 3D space, one can calculate the stresses in these directions using the stress tensor. The proportion of shear and normal stress can be determined by constant coefficients K and B.
% weighting constants for a given criterion
syms K B real
 
% general form of the criterion - weighted sum of shear and normal stress
% on the critical plane
crit = B*St + K*Sn;
 
% we extract the coefficients before the tensor components and write them down to a(i)
sig = [Sxx Syy Szz Sxy Syz Szx];
a = cell2sym(cell(6, 1));
for i = 1:6
k = crit;
for j = 1:6
if j ~= i
k = subs(k, sig(j), 0);
else
k = subs(k, sig(j), 1);
end
end
a(i) = simplify(k);
end
disp(a')
Using symbolic calculations, we obtained 6 mathematical expressions in the variable a. For one position of the critical plane, these are constants which, multiplied by the appropriate members of the stress tensor , give the equivalent stress according to the selected criterion.

2 Setting for the Critical Plane Position

One of the most important elements when applying multiaxial fatigue criteria based on the concept of a critical plane is determining its location. This location depends on the assumptions of the criterion. Sometimes it is the location that gives the greatest damage value according to the criterion assumptions and sometimes it is the location of the greatest shear or normal stress. Please look into the definition of particular criterion to determine this.
Regardless, the common problem of all criteria is the search for a local maximum. The problem is even greater because there may be many areas with a local maximum or one global maximum extending over a certain area.

2.1 Maximum Normal Stress

To show how to proceed, we present a search for possible locations of the critical plane for a given stress state, assuming that this is the plane of maximum normal stress. We will search the area represented by points on a sphere of radius 1.
% opints on the sphere
points = generate_spherical_points(1000);
 
% Extract x, y, and z coordinates from the points
x = points(:, 1);
y = points(:, 2);
z = points(:, 3);
 
% Create a 3D scatter plot
figure
scatter3(x, y, z, 10)
xlabel('x'), ylabel('y'), zlabel('z')
grid on, axis equal
Now we can define the stress state using the components of the tensor. Then we project these components onto the directions indicated by the points and determine the location of the plane where the stress will reach its maximum value.
% stress vector [Sxx Syy Szz Txy Tyz Tzx]
Stress = [100 0 0 0 0 0];
 
% coeficient
a = zeros(length(points), 6);
 
a(:,1) = x.^2;
a(:,2) = y.^2;
a(:,3) = z.^2;
a(:,4) = 2*x.*y;
a(:,5) = 2*y.*z;
a(:,6) = 2*z.*x;
 
SNormalToPlane = a*Stress';
 
[StressMax, SM_ind] = max(SNormalToPlane);
 
% Create a 3D scatter plot
figure
scatter3(x, y, z, 18, SNormalToPlane)
% and vector!!!
xlabel('x'), ylabel('y'), zlabel('z')
grid on, axis equal
